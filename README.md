# Bullish take home


### Requirements
- Java 17
- Maven
### Installation
1. Clone repo
```
  git clone https://github.com/dahcsguy/bullish-take-home.git
```
2. Go to project directory
3. ```mvn clean install```
4. ```java --enable-preview -jar .\target\bullish-0.0.1-SNAPSHOT.jar```
